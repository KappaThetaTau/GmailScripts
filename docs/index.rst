Theta Tau Email Script Docs
==========================================

**This script helps the members of Theta Tau Kappa Chapter communicate in a more personalized way**

------------------------------------------------------------------------------

Welcome to the documentation for the Kappa Chapter's email script.  The repository is located on Github_. These scripts help the chapter communcate both internally and externally in a more personalized way.  In addition to support of multipart emails, it also supports the replacement of strings within an email. See the sections below for more detailed documentation, and if you find any bugs please submit them on the Github issues.
